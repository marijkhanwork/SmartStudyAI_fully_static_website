# 🚀 SparkCrafters - AI-Powered Learning Platform

## 📖 Introduction

**SparkCrafters** is a next-generation AI-powered learning platform that helps students and lifelong learners master any subject. It combines AI technology with proven learning methodologies like spaced repetition and gamification to make learning effective and enjoyable.

> **Live Website:** [https://sparkcrafters.netlify.app/](https://sparkcrafters.netlify.app/)

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| **🤖 AI Card Generator** | Instantly create smart flashcards on any topic. Just describe what you want to learn and AI does the rest. |
| **⚔️ Battle Arena** | Participate in knowledge battles against AI bots. Answer correctly to deal damage and climb the leaderboard! |
| **🔄 Leitner System** | Scientifically proven spaced repetition algorithm that automatically optimizes memory retention. |
| **💬 AI Chat Tutor** | 24/7 personal AI study assistant that answers questions and provides explanations. |
| **📝 MCQ Generator** | Automatically generate multiple-choice questions. Create custom tests or let AI handle it. |
| **📄 PDF Downloads** | Export flashcards and MCQs as professional PDF documents. Perfect for offline study and sharing. |
